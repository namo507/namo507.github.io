## Project Overview

Designed and deployed comprehensive content management system for Indian Red Cross Society Karnataka chapter, streamlining volunteer management and donor engagement.

## Key Contributions

<div class="metric-card">
  <strong>CMS Development:</strong> Designed Drupal-based CMS for volunteer registry and donor database, reducing manual data entry by 50%
</div>

<div class="metric-card">
  <strong>Multilingual Support:</strong> Implemented multilingual support (Hindi/Kannada) achieving 95% bug-free experience across 10,000+ monthly visitors
</div>

<div class="metric-card">
  <strong>Database Integration:</strong> Built custom modules for volunteer registration, event management, and donation tracking with MySQL backend
</div>

## Technologies Used

- **CMS:** Drupal 9
- **Backend:** PHP, MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Features:** Multilingual, Responsive Design

## Key Results

| Metric | Value |
|--------|-------|
| Data Entry Reduction | 50% |
| Monthly Visitors | 10,000+ |
| Bug-free Experience | 95% |
| Languages Supported | 3 (English, Hindi, Kannada) |

## Features Implemented

1. **Volunteer Registry:** Online registration and profile management
2. **Donor Database:** Contribution tracking and receipts
3. **Event Management:** Blood donation camp scheduling
4. **News & Updates:** Dynamic content publishing
5. **Contact Forms:** Inquiry and support system
6. **Multilingual:** Hindi and Kannada translations

## Impact

- Streamlined volunteer coordination for blood donation camps
- Improved donor engagement and retention
- Enhanced organizational visibility and outreach
- Reduced administrative overhead by 40%
