## Project Overview

Conducted comprehensive study on machine learning and IoT applications for concrete strength prediction and quality monitoring in civil engineering applications.

## Key Contributions

<div class="metric-card">
  <strong>Literature Synthesis:</strong> Synthesized 50+ peer-reviewed papers on ML-based concrete compressive strength prediction, identifying Random Forest and XGBoost as top-performing models (R² up to 0.89)
</div>

<div class="metric-card">
  <strong>IoT Integration Framework:</strong> Proposed IoT sensor integration framework for real-time curing environment monitoring, enabling dynamic strength estimation
</div>

<div class="metric-card">
  <strong>Sustainability Analysis:</strong> Evaluated ML approaches for optimizing supplementary cementitious materials (SCMs) usage, projecting 15% cement reduction with maintained structural integrity
</div>

## Technologies Used

- **Languages:** Python
- **ML Models:** Random Forest, XGBoost, SVR
- **IoT:** Sensor Integration Framework
- **Domain:** Concrete Technology, Materials Science
